Одиночка - это паттерн, который гарантирует, что для класса будет создан только один экзмепляр. Этот паттерн используется тогда, когда нам нужно создать только один экземпляр какого-либо класса. Наприме, есть класс Игра. Так как, игра возможна только с одним мячом на площадке, то экземпляр класса Игра может вызвать функцию класса Мяч, которая вводит мяч в игру. При этом, если мяч уже введен в игру, то ввести другой мяч в игру не получится.
