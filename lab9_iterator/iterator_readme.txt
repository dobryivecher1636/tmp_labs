Паттерн Итератор предоставляет абстрактный интерфейс, позволяющий обеспечить последовательный доступ ко всем элементам составного объекта без раскрытия его внутренней структуры. Итератор используется, когда необходимо представить единый интерфейс для перебора объектов из какого-то набора, или, когда нужно предоставить несколько вариантов перехода одного объекта.
